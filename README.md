# 坦克大战 - Tank Battle

一款纯 HTML5 Canvas 的经典坦克大战网页游戏。

## 游戏截图

![Tank Battle](screenshot.png)

## 玩法

- ⬆⬇⬅➡ 方向键移动坦克
- 空格键射击
- R 键重新开始

## 特性

- 10个难度递增的关卡
- 多种地形：砖墙（可摧毁）、钢铁（不可摧毁）、水域（不可通过）、森林（隐蔽）
- 敌人 AI：巡逻 + 追踪玩家
- 道具系统：加速、连发、护盾、炸弹
- 爆炸粒子特效和屏幕震动
- 保护基地（黄色旗帜）

## 快速开始

直接在浏览器中打开 `tank-battle.html` 即可游戏。

## 技术

纯前端实现，使用 HTML5 Canvas API，无需任何依赖或构建工具。
